# Wasaa Family — Legal Documents

A Next.js web application serving the Privacy Policy and Terms of Service for the Wasaa Family platform.

**Pages:**
- `/` — Legal hub with document index and key commitments
- `/privacy` — Privacy Policy
- `/terms` — Terms of Service

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build   # outputs to /out for static hosting
```

Configured for static export (`output: 'export'`), compatible with GitHub Pages, Vercel, or any static host.
